# 🤖 Automação de Contratos em Massa (Prestador de Serviço) PT

## 📌 Visão Geral

Este projeto tem como objetivo automatizar a leitura de dados de uma planilha Excel e a geração de contratos personalizados para prestadores de serviço.  
A automação preenche campos específicos de um modelo de contrato, gera os arquivos finais (ex: PDF ou DOCX) e os salva automaticamente em uma pasta definida pelo usuário.

> **Aviso Ético:** Os dados apresentados neste exemplo são **totalmente fictícios** e foram gerados aleatoriamente para fins de demonstração. Nenhuma informação real de fornecedores ou prestadores foi utilizada.

---

## 🎯 Funcionalidades

- Leitura de planilha Excel (`.xlsx` ou `.xls`)
- Preenchimento automático de contratos a partir dos dados
- Geração de múltiplos contratos em lote (lote em massa)
- Salvamento automático em pasta específica
- Geração em formatos comuns (PDF, DOCX ou texto simples)
- Log de execução para rastreamento

# 🤖 Mass Contract Automation (Service Provider) EN

## 📌 Overview

This project aims to automate reading data from an Excel spreadsheet and generating personalized contracts for service providers.  
The automation fills specific fields in a contract template, generates the final files (e.g., PDF or DOCX), and automatically saves them in a user-defined folder.

> **Ethical Notice:** The data presented in this example are **entirely fictitious** and were randomly generated for demonstration purposes. No real information from suppliers or service providers has been used.

---

## 🎯 Features

- Excel spreadsheet reading (`.xlsx` or `.xls`)
- Automatic contract filling based on data
- Batch generation of multiple contracts (mass batch)
- Automatic saving to a specific folder
- Generation in common formats (PDF, DOCX, or plain text)
- Execution log for tracking
