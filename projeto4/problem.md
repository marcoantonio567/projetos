# Spreadsheet to Web System Automation / Automação de Planilha para Sistema Web

## English

### My Work Routine

Hey friend, I need to share a routine I have here at work. I use an Excel spreadsheet that feeds a web system I access with a username and password.

The spreadsheet is populated with data from the queries I make in the system, and it helps a lot when organizing the information.

The only annoyance is having to manually enter the data every time, but I think with a little automation, this task can become easier.

### Project Notes

**URL Note:** The website shown in the app's URL is not the client's actual system, but it's something similar.

**Personal Reflection:** I really enjoyed working on this project. If I were to improve it someday, I would implement the BeautifulSoup (bs4) library to make it faster and more practical.

## Português

### Minha Rotina de Trabalho

E aí amigo, preciso compartilhar uma rotina que tenho aqui no trabalho. Eu uso uma planilha em Excel que alimenta um sistema web que acesso com usuário e senha.

A planilha é alimentada com os dados das consultas que eu faço no sistema e isso ajuda muito na hora de organizar as informações.

A única chatice é ter que ficar inserindo os dados manualmente toda vez, mas acho que com um pouco de automação, dá pra deixar essa tarefa mais fácil.

### Notas do Projeto

**Observação da URL:** O site que está na URL do app não é o mesmo site do sistema do cliente, mas é algo parecido.

**Reflexão Pessoal:** Pessoalmente gostei muito de fazer esse projeto. Se algum dia eu fosse melhorar ele, eu iria implantar a biblioteca bs4 para ficar mais rápido e mais prático.
