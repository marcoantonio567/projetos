# Signature Script Generator / Gerador de Script de Assinatura

## English

**Request:**\
"Hey Saitama, generate a signature script so that when the client signs up for our system, it automatically creates a signature with the client's data. Here's the base image – build the script on top of this base image."

### Instructions:

1. Use the provided base image as a template.
2. Automatically generate a signature upon client sign-up.
3. Populate the signature with the client's data (e.g., name, date, contract details).
4. Ensure the script integrates with your existing system.

***

## Português

**Solicitação:**\
"Fala Saitama, gera um script de assinatura pra quando o cliente assinar nosso sistema automaticamente já gere uma assinatura com os dados do cliente. Toma aqui a imagem base e constrói o script em cima dessa imagem base."

### Instruções:

1. Use a imagem base fornecida como modelo.
2. Gere automaticamente uma assinatura no momento da assinatura do cliente.
3. Preencha a assinatura com os dados do cliente (ex.: nome, data, detalhes do contrato).
4. Certifique-se de que o script integre-se ao seu sistema existente.
